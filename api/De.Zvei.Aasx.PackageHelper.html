    Remarks

    No new class for the AASX package was defined that inherits from System.IO.Packaging.Package to represent the AASX package. This was not done, because it is required to override some methods of abstract class "Package" and would consume much more efforts to do so. The best solution was to use the Package object returned by Package.Open() and define a set of helper methods to handle that package.
